# Vinylab GitHub Pages 버전

이 폴더의 내용 전체를 `si-yeop/MPIOS` 저장소의 최상위 경로에 업로드하세요.

업로드 후 GitHub 저장소에서 다음 설정을 한 번 확인하세요.

1. `Settings` → `Pages`로 이동합니다.
2. `Build and deployment`의 `Source`를 `GitHub Actions`로 선택합니다.
3. `Actions` 탭에서 `Deploy Vinylab to GitHub Pages` 작업이 완료될 때까지 기다립니다.
4. `https://si-yeop.github.io/MPIOS/`를 새로고침합니다.

템플릿은 서버가 아닌 현재 브라우저에 저장됩니다. 브라우저 데이터 삭제, 시크릿 모드 사용 또는 다른 기기 접속 시 저장 목록이 공유되지 않습니다. 음악 파일은 템플릿에 포함되지 않습니다.
